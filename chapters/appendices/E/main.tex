\section{Code availablility}

All code produced for \cref{part:gpu} of this thesis is licenced under the GNU General Public License (GPL), either version 2 of the License, or any later version.
The interface code implemented in \openqxd can be found in the Gitlab repository~\cite{gitlab:openqxd-devel}, under the branch name \code{main-thesis-release}.
The related interface code in \quda can be found in the GitHub repository~\cite{github:quda:chaoos} under the branch name \code{tree/feature/openqxd}.
Both merge-requests are currently under review.

All code produced for \cref{part:variance} of this thesis is licenced under the GNU General Public License (GPL), either version 2 of the License, or any later version.
The code implemeting the mulrgrid low-mode averaging method in \openqxd can be found in the Gitlab repository~\cite{gitlab:openqxd-devel}, under the branch name \code{34-lma-scheme}.
